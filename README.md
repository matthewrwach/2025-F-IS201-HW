# Final Web Development Project

A professional portfolio website featuring HTML, CSS, and JavaScript demonstrations.

## Project Structure

- `index.html` - Home page with professional introduction
- `resume.html` - HTML résumé with education, work experience, skills, and awards
- `scratch.html` - From-scratch hobby page about space exploration
- `app.html` - Interactive Run 3 endless runner game
- `styles/` - CSS stylesheets
  - `professional.css` - Styles for professional pages
  - `scratch.css` - Custom stylesheet for scratch page
- `images/` - Project images directory

## Features

### Professional Pages
- Clean, modern design with professional appearance
- Responsive navigation
- HTML résumé (not PDF)
- Good color contrast and readability

### Scratch Page
- Custom HTML/CSS from scratch
- Embedded YouTube video
- Interactive Tableau visualization
- Nested lists (ordered list with unordered lists)
- Custom images
- On-page anchor navigation
- Custom background gradient

### Interactive Web App
- Run 3 endless runner game
- Single-page application with embedded styles and scripts
- Score tracking with local storage
- Keyboard controls (Space/Up to jump, Down to slide)

## GitHub Pages Setup

1. Push this repository to GitHub
2. Go to repository Settings > Pages
3. Select the main branch as the source
4. Your site will be available at: `https://[username].github.io/[repository-name]/`

## Local Development

Simply open `index.html` in a web browser to view the site locally.
